# 📊 SQL Data Analysis Project
SQL data analysis project using SQL Server (SSMS)

##  Overview
This project was completed as part of my Data Analysis training at the National Telecommunication Institute (NTI). The goal was to analyze a dataset using SQL Server to extract meaningful insights.

##  Objectives
- Analyze trends and patterns in the dataset  
- Generate insights using SQL queries  

##  Tools & Technologies
- SQL Server (SSMS)

## 🔍 Key Analysis

### 1. Data Exploration
- Understanding the database schema  
- Analyzing table relationships (ERD/diagram)  
- Inspecting data structure and contents  

### 2. Querying & Analysis
- SELECT, FROM, WHERE  
- HAVING clause  
- GROUP BY and ORDER BY  
- Joins (INNER, LEFT, etc.)  
- Grouping and filtering
